How to run this application:

* Ruby version
  - ruby 2.6.5
  - rails 6.0.1

* Configuration
go to directory project and run
```
$ bundle install
```

* Database creation
run
```
$ rails db:reset
```

* Database initialization
```
$ rails db:migrate
```

* How to run the test suite
```
$ bundle exec rspec
```
